# Contributing

Thanks for your interest in contributing! To keep contributions smooth and consistent, please follow these guidelines.

- **Report bugs:** Open a clear issue with steps to reproduce and expected behavior.
- **Propose changes:** Open a pull request (PR) against the `main` branch with a concise description of what you changed and why.
- **Code style:** Follow the existing code style in the repository. Keep changes focused and atomic.
- **Tests:** Add or update tests for any new behavior where applicable.
- **Commits:** Use clear commit messages and squash or rebase when appropriate.

If you have questions before contributing, open an issue and we'll help you get started.
